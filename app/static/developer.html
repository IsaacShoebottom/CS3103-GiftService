Present Registry Web Service

This documentation lists the endpoints for app, so that developers can write their own awesome applications that use the app webservice!

/presents/<string:username>

GET: Returns a json document containing the all presents for the specified username.

Example: /presents/admin results in

      {
        "presents": [
          {
            "link": "example.com",
            "presentId": "1",
            "title": "Test"
          },
          {
            "link": "example2.com",
            "presentId": "2",
            "title": "Test2"
          },
          
          etc...
      }
